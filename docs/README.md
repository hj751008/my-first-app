# Suji Math AI Docs Map

## Purpose
- Keep root documents short and stable.
- Move session logs, design decisions, and long-range planning into focused folders.
- Make the next session easier to resume without rereading one large file.

## Structure
- `docs/handoffs/`
  - Session-by-session working notes and resume points.
- `docs/decisions/`
  - Durable design decisions and operating rules.
- `docs/roadmap/`
  - Expansion plans across grades, units, and milestones.

## Root Files
- `PROJECT_CONTEXT.md`
  - Current project facts only.
- `NEXT_STEPS.md`
  - Short active task list only.

## Reading Order For A New Session
1. `PROJECT_CONTEXT.md`
2. `NEXT_STEPS.md`
3. The latest file in `docs/handoffs/`
